üêô GitHub Sign-in Setup

üîß Setup Steps:

  • Create OAuth App: Go to GitHub ‚Üí Settings ‚Üí Developer settings ‚Üí OAuth Apps
  • New OAuth App: Click "New OAuth App"
  • Configure App:
    • Application name: Your app name
    • Homepage URL: Your app URL
    • Authorization callback URL: https://your-project.firebaseapp.com/__/auth/handler
  • Get Credentials: Copy Client ID and Client Secret
  • Enable in Firebase: Go to Firebase Console ‚Üí Authentication ‚Üí Sign-in method
  • Configure GitHub: Enable GitHub provider and add your credentials
üîó Callback URL: The callback URL format is always https://PROJECT_ID.firebaseapp.com/__/auth/handler
